Application goals

Goal Description
Solving the nuclear decay problem Application must be able to solve the nuclear decay problem in numerical and analytical way.
Comparing Runge-Kutta and Runge-Kutta-Fehlberg methods Application must be able to compare Runge-Kutta (RK) and Runge-Kutta-Fehlberg (RKF) methods based on nuclear decay problem solved in analytical way.
Visualization of output data Application must have a module to visualize the nuclear decay problem solution and comparison of RK and RKF methods.
Saving the output data Application must have a module to save both data of nuclear decay problem solution and the data of RK and RKF comparison to proper files.

Functional requirements

Requirement Description
Implementation of Runge-Kutta methods Application has implemented Runge-Kutta methods of 1st, 2nd, 3rd, 4th, 5th and 6th order.
Implementation of Runge-Kutta-Fehlberg methods Application has implemented Runge-Kutta-Fehlberg methods of 5th, 6th, 7th and 8th order.
Data visualization module Module for viewing output data on charts.
Saving module Module for saving output data to file.

Non-functional requirements

Requirement Description
Language Application Interface is created in English language.
Operating system Application is created on Windows 10 OS.
